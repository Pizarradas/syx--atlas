# Knowledge Front-end — Índice

Fundamentos de implementación front-end que respaldan las reglas técnicas de los modos SYX.

---

## Módulos

| Módulo | Qué aporta |
|--------|-----------|
| `html-semantics.md` | Elementos semánticos, jerarquía de headings, roles ARIA base |
| `css-architecture.md` | Cascada CSS, especificidad, @layer, progressive enhancement CSS |
| `mobile-first.md` | Worst-case first, breakpoints min-width, DOM order, clamp() |
| `progressive-enhancement.md` | Tres capas HTML/CSS/JS independientes y por qué |
| `accessibility-wcag.md` | WCAG 2.1/2.2 AA, ARIA, gestión del foco, criterios aplicados |
| `javascript-patterns.md` | Patrones JS para interactividad accesible en SYX |
| `size-models.md` | Modelos matemáticos para escalas de tipo y espaciado: modular, áurea, métrica, progresión aritmética de espaciado y relación tipografía/espaciado |
| `size-models-checklist.md` | Checklist de validación de coherencia de escala: base documentada, fórmula explícita, tokens indexados, sin snowflakes |
