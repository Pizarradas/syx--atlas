# Mapa del sistema documental GSAP

## Objetivo

Este sistema sirve para nombrar efectos de animación con precisión, especialmente cuando la intuición visual existe pero el término técnico o documental todavía no está claro.

## Flujo de consulta

1. Empezar en `01-fundamentos/` para alinear el modelo mental de GSAP como motor temporal y narrativo.
2. Ir a `02-capacidades/` para entender herramientas como tween, timeline, stagger y ScrollTrigger.
3. Consultar `03-patrones/` para elegir un nombre operativo del efecto visual que se quiere producir.
4. Usar `04-glosario/` para prompts y nomenclatura corta.
5. Duplicar una plantilla de `05-plantillas/` para crear nuevos microdocs.
