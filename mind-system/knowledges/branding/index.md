# Knowledge Branding — Índice

Fundamentos de percepción de marca, credibilidad y prestigio visual. Respalda las decisiones de CREATIVE, UX y AUDIT cuando el objetivo es comunicar autoridad, confianza o posicionamiento premium.

---

## Módulos

| Módulo | Qué aporta |
|--------|-----------|
| `perception-of-prestige-foundations.md` | Psicología cognitiva del prestigio: modelo PRI, processing fluency, heurísticos de Cialdini, semiótica del lujo |
| `perception-of-prestige.rules.md` | 17 reglas operativas (R-TYP, R-SPC, R-COL, R-SYS, R-AUT, R-DET, R-ESC) con checks y output template para auditoría |

---

## Relación con los modos

| Regla / decisión | Knowledge de origen |
|------------------|---------------------|
| Espacio en blanco como señal de dominio | `perception-of-prestige-foundations.md` → vectores comunicativos |
| Tipografía serif vs geométrica según registro | `perception-of-prestige-foundations.md` → semiótica del lujo |
| Auditoría de densidad, jerarquía, consistencia | `perception-of-prestige.rules.md` → bloques 1–4 |
| Auditoría de autoridad, prueba social, lenguaje | `perception-of-prestige.rules.md` → bloque 5 |
| Auditoría de contención y exclusividad | `perception-of-prestige.rules.md` → bloque 7 |
